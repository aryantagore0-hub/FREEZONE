# 🚀 VortexFrag 3D - Ultimate Browser FPS Battle Royale

**VortexFrag 3D** is a high-octane, browser-based First-Person Shooter (FPS) game built with modern web technologies. Experience intense battle royale action directly in your browser with no downloads required!

## 🎮 Play Now
[Click here to play VortexFrag 3D](https://aryantagore0-hub.github.io/FREEZONE/)

## ✨ Key Features
- **3D FPS Gameplay:** Realistic shooting mechanics and 3D environment.
- **Multiplayer Mode:** Create or join rooms using PeerJS for real-time combat with friends.
- **Battle Royale Style:** Be the last one standing in the arena.
- **Optimized for Web:** Lightweight and runs smoothly on most modern browsers.
- **Health & Inventory:** Use Medkits and manage your ammo to survive.

## ⌨️ Controls
| Action | Key |
| :--- | :--- |
| **Move** | W, A, S, D |
| **Shoot** | Left Click |
| **Heal** | H (Medikit) |
| **Reload** | R |
| **Jump** | Space |

## 🛠️ Tech Stack
- **Frontend:** HTML5, CSS3 (Styling & HUD)
- **Engine:** JavaScript (Custom 3D Physics & Logic)
- **Multiplayer:** PeerJS (WebRTC)
- **Hosting:** GitHub Pages
